Strabo Spot Mobile is a part of a larger Data System for Structural Geology and Tectonics (SG&T). It consists of a database, a web site and a mobile app. The entire Strabo Spot project is still in development and as such the mobile app is not yet in the Google Play Store or the iOS App Store but can be tested by accessing it as described below.

Current Version 0.17.8

Accessing the Main Site
http://strabospot.org/

Downloading the Mobile App
http://strabospot.org/download

**Note for iOS users:**
The link above will not work for iOS devices unless your device has been previously registered as a valid testing device for this particular project. The app can be installed on iOS devices through the use of the Ionic View app. Search the App Store for Ionic View. Preview a Shared App and use the id fbd0e989. Some functionality is limited when using the app through Ionic View however, such as the camera/photos.

A web version of the app can be accessed at https://app.strabospot.org/. Again, as above, functionality is limited when using the app this way.

Bugs, Feature Requests and Other Issues
- The preferred method of reporting is to post an issue at https://github.com/StraboSpot/strabo-mobile/issues. Please post one issue per bug/issue/feature request and include the version number.
- Alternatively, you can send an email to strabospot@gmail.com.
